# loquacious-octo-tanuki
Oluiden arvostelujärjestelmä toteutettuna Node.js:llä

##Ominaisuuksia
- Lisää olut
- Listaa oluet
- Listaa yksittäinen olut arvosteluineen
- Poista olut
- Kirjoita arvostelu yksittäiselle oluelle
